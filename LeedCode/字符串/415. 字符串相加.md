# 415. 字符串相加
## 描述
给定两个字符串形式的非负整数 num1 和num2 ，计算它们的和并同样以字符串形式返回。    
你不能使用任何內建的用于处理大整数的库（比如 BigInteger）， 也不能直接将输入的字符串转换为整数形式。             

### 示例
示例1   
> 输入：num1 = "11", num2 = "123"         
> 输出："134"       

示例2   
> 输入：num1 = "456", num2 = "77"                     
> 输出："533"          

示例3   
> 输入：num1 = "0", num2 = "0"            
> 输出："0"

## 解决
### 思路一: 模拟「竖式加法」的过程
![竖式加法](https://img-blog.csdnimg.cn/img_convert/c413b5120b31683204719224a5e236d3.png)

```javascript
/**
 * @param {string} num1
 * @param {string} num2
 * @return {string}
 */
var addStrings = function (num1, num2) {
  let i = num1.length - 1,
    j = num2.length - 1,
    add = 0;  // 表示进位
  const ans = [];
  while (i >= 0 || j >= 0 || add != 0) {
    const x = i >= 0 ? num1[i] - "0" : 0;
    const y = j >= 0 ? num2[j] - "0" : 0;  // 转成数字
    const result = x + y + add;
    ans.push(result % 10);
    add = Math.floor(result / 10);
    i -= 1;
    j -= 1;
  }
  return ans.reverse().join("");
};
```
时间复杂度： O(max(len1​,len2​))       
空间复杂度： O(1)   
**不使用数组：**
```javascript
/**
 * @param {string} num1
 * @param {string} num2
 * @return {string}
 */
var addStrings = function (num1, num2) {
  let i = num1.length,
    j = num2.length,
    result = '', 
    add = 0;  // 表示进位
  while (i || j) {
    i ? add += +num1[--i]: '';
    j ? add += +num2[--j]: '';

    result = add % 10 + result;
    if (add > 9) add = 1
    else add = 0
  }
  if(add) result = 1 + result;
  return result
};
```
